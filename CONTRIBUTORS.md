# Contributors

a14y is an open spec, a versioned scorecard, and a CLI plus Chrome extension
that score how cleanly any website can be read by AI agents. It gets better
because of the people below: folks who write checks, edit the docs, sharpen the
scorecard, and stress-test the spec. Thank you.

- **Timothy Jordan** ([@timothyjordan](https://github.com/timothyjordan)):
  creator and maintainer. The CLI, the `@a14y/core` library, the Chrome
  extension, the scorecard, and the docs site.
- **Jesse Robbins** ([@jesserobbins](https://github.com/jesserobbins)): the
  CONTRIBUTING.md guide and AGENTS.md ([#20](https://github.com/timothyjordan/a14y/pull/20)),
  plus spec and scorecard work including modified-date check correctness
  ([#21](https://github.com/timothyjordan/a14y/issues/21)) and docs-first
  ordering for scorecard PRs ([#30](https://github.com/timothyjordan/a14y/issues/30)).

## Adding yourself

Contributed something? Add yourself in the same PR. Append a bullet with your
name, your GitHub handle, and a short note on what you contributed (link the PR
or issue). Keep the list alphabetical by last name after the maintainer entry.

For branching, commit, and PR conventions, see
[CONTRIBUTING.md](./CONTRIBUTING.md).
