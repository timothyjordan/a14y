# Single purpose

Single-purpose description for the Chrome Web Store upload form.

---

```
a14y has exactly one purpose: score a website for how readable it is
by AI agents, using the open a14y scorecard. The popup presents two
actions — check the current page, or scan the whole site — and
displays a 0–100 score with a per-check breakdown. No analytics, no
data collection, no cross-site tracking; every audit runs entirely
in the user's browser.
```
